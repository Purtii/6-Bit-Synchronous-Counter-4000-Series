Project: 6-Bit Synchronous Counter

The objective of this project is to design and implement a 6-bit synchronous counter using dedicated logic components. The design adheres to the following technical specifications:
	- Architecture: Fully synchronous counting logic to ensure simultaneous state transitions across all bits.
	- Technology: Exclusively built using CMOS 4000 series integrated circuits (bistables/flip-flops and logic gates).
	- Clock Sources: The system features a versatile oscillator unit providing three distinct clock signals: 
		- 2Hz: Low-frequency square wave for real-time visual monitoring.
		- 200kHz: High-frequency square wave for performance testing.
		- Manual Trigger: A debounced manual pulse for step-by-step state verification.
	- Interface: * 6-LED Array: Direct binary representation of the counter's output (Q0 to Q5).
	- Reset: Dedicated master reset button to initialize the counter to 000000.
